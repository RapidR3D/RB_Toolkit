# <color=#FFD700>Add Work & New Work</color>  

## <color=#FF8C00>Overview</color>  

**Add Work** and **New Work** allow you to handle unplanned car movements that weren't on your original work order.  

---

## <color=#FF8C00>Add Work vs. New Work</color>  

### <color=#20B2AA>Add Work</color>  
<color=#FF8C00>**Definition:**</color> You initiate the addition of work  
<color=#FF8C00>**Use when:**</color> You need to move cars not on your work order  
<color=#FF8C00>**Process:**</color> Crew-initiated, manual input  

<color=#FF8C00>**Examples:**</color>  
- Empty pulls from customer (not pre-planned)
- Bad order cars
- Cars found at wrong location
- Emergency moves

### <color=#20B2AA>New Work</color>  
<color=#FF8C00>**Definition:**</color> System sends you additional work  
<color=#FF8C00>**Use when:**</color> Dispatched new assignments during your shift  
<color=#FF8C00>**Process:**</color> System-initiated, sent via notification  

<color=#FF8C00>**Examples:**</color>  
- New customer orders received
- Work reassigned from another crew
- Updated plans from dispatcher
- Additional interchange cars

## <color=#FF8C00>Add Work Process</color>  

### <color=#20B2AA>Step 1: Select FROM Instruction</color>  

<color=#FF8C00>**Available Options:**</color>  
- **PK** (Pick Up from Station)
- **PU** (Pull from Customer)
- **IP** (Intra Plant Switch)
- **OS** (Off Spot)
- **LC** (Pick Up Loco/EOT)
- **WI** (Weigh In Industry)
- **TU** (Turn)
<color=#FF8C00>**Choose the instruction that describes where you GOT the car from**</color>  
---

### <color=#20B2AA>Step 2: Select TO Instruction</color>  

<color=#FF8C00>**Available Options:**</color>  
- **SO** (Station Set Out)
- **DI** (Deliver Interchange)
- **PL** (Place at Customer)
<color=#FF8C00>**Choose the instruction that describes where you're TAKING the car to**</color>

## <color=#FF8C00>Handling New Work</color>  

### <color=#20B2AA>Decision: Accept or Reject</color>  

<color=#FF8C00>**Accept:**</color> You CAN handle this additional work  
<color=#FF8C00>**Reject:**</color> You CANNOT or will not perform this work  

### <color=#20B2AA>Accepting New Work</color>  

<color=#FF8C00>**Step 1:**</color> Review the new work cars  

<color=#FF8C00>**Step 2:**</color> Select cars you can handle  
- Tap individual cars
- Use **SELECT/DESELECT ALL**
- Use **Select Range**

<color=#FF8C00>**Step 3:**</color> Tap **ACCEPT SELECTED CARS**  

<color=#FF8C00>**Step 4:**</color> Cars added to your work order with:  
- FROM instruction (where to get car)
- TO instruction (where to take car)
- Location details

<color=#FF8C00>**Step 5:**</color> Report work when performed  
- New cars appear in appropriate tiles
- Report like any other work

---

### <color=#20B2AA>Rejecting New Work</color>  

<color=#FF8C00>**Step 1:**</color> Review the new work cars  

<color=#FF8C00>**Step 2:**</color> Select cars you CANNOT handle  

<color=#FF8C00>**Step 3:**</color> Tap **REJECT SELECTED CARS**  

<color=#FF8C00>**Step 4:**</color> System removes from your assignment  

<color=#FF8C00>**Purpose of Rejecting:**</color>  
- Clears notification/message
- Informs system you won't perform work
- Allows reassignment to another crew

<color=#FF8C00>**When to Reject:**</color>  
- Out of time
- Wrong location
- Equipment constraints
- Safety concerns
- Received in error

## <color=#FF8C00>Common Add Work Scenarios</color>  

### <color=#20B2AA>Scenario 1: Emergency Empty Pull</color>  

1. Customer calls: Empty available for pickup  
2. Tap ADD WORK  
3. Select PU (Pull from Customer)  
4. Choose customer  
5. Select empty car from inventory  
6. Set TO: SO (Set Out at yard)  
7. Select destination yard  
8. Add to work order  
9. Pull car when at customer  
10. Report pull work with actual time  

### <color=#20B2AA>Scenario 2: Bad Order Discovery</color>  

1. Find bad order car at customer  
2. Tap ADD WORK  
3. Select PU (Pull from Customer)  
4. Choose customer  
5. Select bad order car  
6. Set TO: SO (Set Out at yard)  
7. Select yard with repair track  
8. Add to work order  
9. Pull car  
10. Report with BO exception code  

### <color=#20B2AA>Scenario 3: Found Car at Wrong Location</color>  

1. Find car at location, not on work order  
2. Verify with dispatcher  
3. Tap ADD WORK  
4. Select PK (Pickup from Station) or PU (Pull from Customer)  
5. Select car from inventory  
6. Set TO instruction  
7. Select destination  
8. Add to work order  
9. Move car to correct location  
10. Report work  

---

## <color=#FF8C00>Common New Work Scenarios</color>  

### <color=#20B2AA>Scenario 1: Customer Releases Load</color>  

1. Notification bell shows NEW WORK  
2. Tap notification  
3. Review: Customer has 2 loaded pulls  
4. Verify you can handle (not out of time)  
5. Select both cars  
6. Tap ACCEPT SELECTED CARS  
7. Cars added to customer PU tile  
8. At customer, pull the loads  
9. Report pulls with actual time  

### <color=#20B2AA>Scenario 2: Reassigned Interchange</color>  

1. Notification: 3 cars for interchange delivery  
2. Review New Work  
3. Verify destination railroad  
4. Assess time to complete  
5. Accept all 3 cars  
6. Cars added to DI tile  
7. Deliver to interchange  
8. Report DI work  

### <color=#20B2AA>Scenario 3: Cannot Handle New Work</color>  

1. Notification: 5 new cars for distant customer  
2. Review: Would exceed hours of service  
3. Select all 5 cars  
4. Tap REJECT SELECTED CARS  
5. System clears assignment  
6. Work reassigned to another crew  
7. Notification cleared

## <color=#FF8C00>Troubleshooting Add Work & New Work</color>  

### <color=#20B2AA>Cannot Find Car in Inventory</color>  
<color=#FF8C00>**Problem:**</color> Car expected but not showing  
<color=#FF8C00>**Solution:**</color>  
1. Verify correct customer/station selected  
2. Use SEARCH CAR function  
3. Check if car already on another work order  
4. Refresh inventory  
5. Contact yardmaster to verify car location  
6. Call MRT Helpdesk if still not found  

### <color=#20B2AA>Wrong Destination Added</color>  
<color=#FF8C00>**Problem:**</color> Added car with incorrect TO instruction  
<color=#FF8C00>**Solution:**</color>  
1. Before reporting: Can remove and re-add  
2. After reporting: Call MRT Helpdesk to correct  
3. May need to add corrective movement  

### <color=#20B2AA>New Work Not Showing</color>  
<color=#FF8C00>**Problem:**</color> Notified of new work but can't find it  
<color=#FF8C00>**Solution:**</color>  
1. Check notification bell  
2. Look for NEW WORK on customer tiles  
3. Logout and login to refresh  
4. Call MRT Helpdesk if still missing  

### <color=#20B2AA>Accepted New Work by Mistake</color>  
<color=#FF8C00>**Problem:**</color> Accepted work you cannot handle  
<color=#FF8C00>**Solution:**</color>  
1. Immediately exception the cars  
2. Use appropriate exception code/reason  
3. Contact dispatcher  
4. Call MRT Helpdesk if needed
