# <color=#FFD700>Vigilance Test Procedure</color>

1. Select forward or reverse on the control.  
2. Press the vigilance switch.  
3. Select Coast B.  
4. Confirm that the vigilance alarm sounds after approximately 50 seconds.  
5. Confirm that a full-service penalty brake application occurs after approximately 60 seconds.  
6. Recover from the full-service penalty application according to local instructions.  
7. Repeat the test for each operator control unit linked to the remote control locomotive or platform.
