# Locomotive Hand Brake Test Procedure

1. Apply the hand brake on all units in the consist.
2. Release both the independent and automatic brake overrides.
3. Place the direction select toggle switch in forward or reverse.
4. Press either vigilance switch and place the speed select lever in COAST.
5. Observe for one minute, adding at least four seconds for each locomotive in the consist, with the air brakes released:
   - If no movement occurs after the observation period, the hand brakes are sufficient.
   - If movement occurs, the hand brakes are not sufficient. Stop the movement by moving the speed select lever to STOP and adjust the hand brakes as necessary.
