# <color=#FFD700>Print Work Order</color>  
<color=#FF8C00>**Mainframe**</color>  
1. Open up Mainframe on your tablet
2. Put in your <color=#20B2AA>**EMPLOYEE ID:**</color>  and  <color=#20B2AA>**PIN:**</color>
3. Select option <color=#20B2AA>**(27 - T&E MISC FUNCTIONS)**</color>
4. On the next menu select <color=#20B2AA>**(16 - WORK ORDER PRINT)**</color>
5. Option will remain <color=#20B2AA>**INQU**</color>
6. Enter your <color=#20B2AA>**TRAIN ID**</color> example: m50628
7. Enter the <color=#20B2AA>**MP:**</color> of the station you are currently at (example: okc110) NOTE: You can put all 000000(zeros) in for MP: and it should bring up all work orders in the system for that job
8. On the far left underneath the blue letters AC, put the letter <color=#20B2AA>**p**</color> for print
9. Look at the printer you want to use and find the <color=#FF8C00>three letter code</color> (example adk--for the North Crew room)
10. On the right you will see the word <color=#20B2AA>**PRINTERS**</color>, with a <color=#20B2AA>**1**</color> and <color=#20B2AA>**2**</color> underneath
11. On the line underneath the number 1, put the 3 letter printer code
11. Submit by hitting <color=#20B2AA>**return**</color> or <color=#20B2AA>**enter**</color>
12. After submitting you should see a message at the bottom of your screen saying <color=#FF8C00>"Print Successfully Submitted"</color>

- <color=#FF8C00>**Note:**</color> After entering a code for a printer in a location the next time you need to print a work order from the same location the printer code should already be populated. 
- <color=#FF8C00>3 letter printer codes may be a mixture of letters and numbers</color>
