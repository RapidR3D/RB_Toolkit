# <color=#FFD700>Standing Locomotive Air Brake Test Procedure</color>

1. Ensure that the handbrake has been released.  
2. Place the independent brake override lever in FULL.  
3. Place the direction select toggle switch in either forward or reverse.  
4. Press either vigilance switch and place the speed select lever in the COUPLE position. Confirm that the corresponding LED turns green.  
5. Observe that the locomotive does not move when electrical output is produced.  
6. Place the speed select lever in the STOP position.  
7. If the locomotive did not move, the test is complete. If the locomotive moved, immediately place the speed select lever in STOP and apply the locomotive handbrake. Do not use the locomotive until the problem has been corrected.
